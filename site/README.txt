Mia&Paper temporary website upload
Created: 2026-04-30 11:34 UTC

How to use:
1. In cPanel File Manager, open:
/home/currwkdi/miaandpaper.com

2. Upload miaandpaper_site_upload.zip there.

3. Extract the ZIP inside that folder.

4. After extraction, index.html and styles.css should be directly inside:
/home/currwkdi/miaandpaper.com

Correct:
  /home/currwkdi/miaandpaper.com/index.html
  /home/currwkdi/miaandpaper.com/styles.css

Wrong:
  /home/currwkdi/miaandpaper.com/miaandpaper_site_upload/index.html

5. Visit:
http://miaandpaper.com

6. After Namecheap installs SSL, test:
https://miaandpaper.com
https://www.miaandpaper.com

Then enable Force HTTPS Redirect in cPanel > Domains.
